Semantic HTML + structured data = better search rankings and richer previews. Learn to mark up content for search engines, social media, and assistive technology.

## Semantic HTML Foundations

```html
<!-- ❌ Unsemantic: divs everywhere -->
<div>
  <div class="title">Article Title</div>
  <div class="date">2025-04-14</div>
  <div class="content">Article text...</div>
</div>

<!-- ✅ Semantic: search engines understand structure -->
<article>
  <h1>Article Title</h1>
  <time datetime="2025-04-14">April 14, 2025</time>
  <div class="content">Article text...</div>
</article>
```

## Schema.org Microdata

Structured data helps search engines and social networks understand content:

```html
<article itemscope itemtype="https://schema.org/NewsArticle">
  <h1 itemprop="headline">Breaking News: New React Features</h1>

  <meta itemprop="image" content="https://example.com/image.jpg" />
  <time itemprop="datePublished" datetime="2025-04-14">Apr 14</time>
  <span itemprop="author">Jane Doe</span>

  <p itemprop="description">A summary of the article...</p>
  <div itemprop="articleBody">Full article content here...</div>
</article>
```

## Open Graph (Social Media)

```tsx
// In Next.js metadata
export const metadata = {
  title: "How to Build Scalable Web Apps",
  description: "Best practices for production systems",
  openGraph: {
    type: "article",
    title: "How to Build Scalable Web Apps",
    description: "Best practices for production systems",
    images: [
      {
        url: "https://example.com/og-image.jpg",
        width: 1200,
        height: 630,
        alt: "Article preview",
      },
    ],
    authors: ["Jane Doe"],
    publishedTime: "2025-04-14T08:00:00Z",
  },
  twitter: {
    card: "summary_large_image",
    title: "How to Build Scalable Web Apps",
    description: "Best practices for production systems",
    images: ["https://example.com/og-image.jpg"],
    creator: "@janedoe",
  },
};
```

## JSON-LD (Recommended Format)

```tsx
export function BlogPostSchema({ post }) {
  const schema = {
    "@context": "https://schema.org",
    "@type": "BlogPosting",
    headline: post.title,
    description: post.description,
    image: post.imageUrl,
    datePublished: post.publishedAt,
    dateModified: post.updatedAt,
    author: {
      "@type": "Person",
      name: post.authorName,
      url: post.authorUrl,
    },
  };

  return (
    <script
      type="application/ld+json"
      dangerouslySetInnerHTML={{ __html: JSON.stringify(schema) }}
    />
  );
}
```
